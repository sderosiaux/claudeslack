Up and running in 3 steps

No complicated setup. Just simple commands.

1

Create a Slack App

Go to api.slack.com/apps and create your personal bot with Socket Mode enabled.

Enable Socket Mode + Bot Scopes Get your xoxb- and xapp- tokens
2

Run Setup

One command configures everything: Slack connection, Claude hook, and background service.

claude-code-slack-anywhere setup xoxb-TOKEN xapp-TOKEN
3

Start Coding

Navigate to any project and run the command. That's it. You're ready to code from anywhere.

cd ~/myproject && claude-code-slack-anywhere
